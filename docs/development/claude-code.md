# Claude Code

I have started with a very, very basic and rudimentary skill set for my buddy. Even though it is light-years far from being perfect, my rationale is that any skills for Claude is better than no skills at all. So use it if you like, and help to improve it if you find the time. Claude deserves it.

You can find it at <https://github.com/vanWittlaer/sw6-skills>. To use it in your project, type the following commands in Claude's prompt:

```bash
/plugin marketplace add vanWittlaer/sw6-skills
/plugin install sw6-skills@sw6-skills
```

Still, you'll have to remind him every now and then to use the skills he has instead of reinventing the wheel.
